# Journal App (TypeScript)

A client-side journal application built with TypeScript that allows users to create, manage, and filter personal entries based on mood.

---

## Features

- Add journal entries with title, content, and mood
- Delete entries
- Filter entries by mood
- Persistent storage using `localStorage`
- Strong type safety using TypeScript (interfaces, enums, generics)
- Modular architecture (separation of logic, storage, and UI)

---

## Key Concepts Used (covering Ts)

- Interfaces (`JournalEntry`)
- Enums (`Mood`)
- Type Aliases (`Journal`)
- Generics (`findByProperty`)
- Type Assertions
- Modular TypeScript architecture
